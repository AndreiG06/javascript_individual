# 🔐 Генератор паролей

Простое веб-приложение для генерации надёжных паролей на чистом JavaScript (Vanilla JS, ES6+ модули). Без сборщиков и фреймворков.

## Цель проекта

Закрепить практические навыки работы с DOM, событиями, валидацией ввода, массивами объектов и модульной структурой кода в JavaScript — в рамках индивидуальной работы по дисциплине «JavaScript».

## Основные функции

- **Генерация паролей** с настраиваемой длиной (4–64 символа).
- **Гибкая настройка состава**: заглавные, строчные, цифры, спецсимволы.
- **Исключение похожих символов** (`l`, `1`, `I`, `O`, `0`, `o`) для повышения читаемости.
- **Криптографически стойкая случайность** через `crypto.getRandomValues` с защитой от modulo bias.
- **Гарантия наличия выбранных классов символов** — в пароле всегда будет минимум один символ каждого включённого набора.
- **Оценка надёжности** (слабый / средний / надёжный) с визуальным индикатором.
- **История паролей** в `localStorage` (последние 50 записей):
    - поиск по подстроке;
    - фильтрация по уровню надёжности;
    - сортировка по дате, длине, надёжности;
    - копирование и удаление отдельных записей;
    - полная очистка истории.
- **Копирование в буфер обмена** через Clipboard API с фоллбэком для `file://`.
- **Валидация ввода** с понятными сообщениями об ошибках.
- **Адаптивный интерфейс** с переключением вкладок и toast-уведомлениями.

## Установка и запуск

Зависимостей нет — нужен только современный браузер (Chrome / Firefox / Edge / Safari).

### Вариант 1: через локальный сервер (рекомендуется)

ES-модули (`<script type="module">`) требуют HTTP, а не `file://`.

```bash
# Если установлен Python 3:
python -m http.server 8000

# Если установлен Node.js:
npx serve .
```

Откройте в браузере: <http://localhost:8000>

### Вариант 2: расширение для VS Code

Установите **Live Server** и запустите `index.html` через `Open with Live Server`.

### Вариант 3: прямое открытие файла

Можно открыть `index.html` двойным кликом — приложение спроектировано с фоллбэком копирования для `file://`. В отдельных браузерах модули могут не загрузиться напрямую с диска, тогда используйте вариант 1 или 2.

## Использование

1. На вкладке **Генератор** настройте длину и набор символов.
2. Нажмите **Сгенерировать** — пароль появится в верхнем поле, рядом отобразится индикатор надёжности.
3. Кнопка 📋 копирует пароль в буфер обмена.
4. Перейдите на вкладку **История**, чтобы найти, отсортировать или удалить ранее сгенерированные пароли.

## Технические особенности

- ES6+: модули `import`/`export`, стрелочные функции, деструктуризация, шаблонные строки, `const`/`let`, `crypto.randomUUID`.
- Строгая модульная архитектура: чистая логика генерации отделена от UI и хранилища.
- Алгоритм Фишера-Йетса для перемешивания обязательных символов.
- Защита от modulo bias при выборе случайного индекса.
